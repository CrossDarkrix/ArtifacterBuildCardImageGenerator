# ArtifacterImageGen

原神のキャラクターステータスを画像化

## 使い方

```
python3 ArtifacterImageGenerator.py
```

## GUI版の画像出力先

Windowsなら「%USERPROFILE%\ArtifacterImageOutput」

Linuxなら「$HOME/ArtifacterImageOutput」
